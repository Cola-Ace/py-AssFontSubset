# py-AssFontSubset
A lightweight Gradio GUI for subsetting ASS subtitle fonts using FontTools. Reduce font file size by embedding only the characters actually used in your subtitles.

Usage:
`python3 subtitle_subsetter.py`

You need to install <https://github.com/fonttools/fonttools> and <https://github.com/gradio-app/gradio> before you use it.
